# dotfiles
These are all of my dotfiles for my customized Caelestia shell. Please modify them according to your own prefrences. you need to have installed caelestia shell first so you can you use these. This doesn't include the app launcher. Also for discord and spotify i use vesktop and spicetify which you can rice. keep ricing.

Caelestia.theme.css is theme for vesktop/discord.
Also for cursor theme I used Nier-cursor-bin.
